\subsection{Infraestructura Experimental}

Para asegurar que el experimento sea repetible a continuación se detalla el entorno de hardware y software en el cual se ejecutaron las pruebas, la tarea fue realizada en una computadora con las siguientes características de hardware:

\begin{itemize}
    \item \textbf{Procesador:} Intel Core i5-12400F, 2.5 GHz
    \item \textbf{Memoria RAM:} 32 GB DDR4.
    \item \textbf{Almacenamiento:} SSD de 1 TB.
    \item \textbf{Sistema Operativo Host:} Windows 11 Pro.
\end{itemize}

Para la compilación y ejecución de los programas se utilizó el Subsistema de Windows para Linux (WSL), con el siguiente entorno:

\begin{itemize}
    \item \textbf{Distribución:} Ubuntu 24.04.1 LTS.
    \item \textbf{Compilador:} \texttt{g++} (Ubuntu 13.3.0-6ubuntu2~24.04.1) 13.3.0.
\end{itemize}

En todo momento del funcionamiento del programa la computadora estuvo dedicada solo a eso, con Visual Studio Code y la consola abierta corriendo los algoritmos.
